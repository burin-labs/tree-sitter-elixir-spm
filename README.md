# tree-sitter-elixir-spm

Swift Package Manager wrapper for the [tree-sitter-elixir](https://github.com/elixir-lang/tree-sitter-elixir) grammar.

## Usage

Add this package to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/burin-labs/tree-sitter-elixir-spm.git", from: "0.1.0")
]
```

Then add `TreeSitterElixir` as a dependency of your target.
